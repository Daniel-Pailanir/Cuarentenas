# Cuarentenas
Información a nivel comunal y diario de las cuarentenas impuestas en Chile.
Esta información se ha digitalizado utilizando los reportes que realiza el Ministerio de Salud. La información contenida puede diferir por concepto de horas en la aplicación de una cuarentena, por lo cual se pide utilizar bajo ese cuidado e informar si estos datos difiere sustancialmente con la realidad para estar en sintonía.
